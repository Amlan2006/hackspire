## Usage

1. **Enter Policy Scenario**: Type your oceanographic policy scenario
2. **Select Rounds**: Choose 1-3 debate rounds
3. **Start Simulation**: Click to begin the multi-agent debate
4. **View Results**: See real-time blockchain governance results

## Quick Examples

The UI includes 5 pre-loaded example scenarios:
- Offshore wind farm development
- Blockchain carbon credits
- Deep-sea mining regulations
- Marine sanctuary establishment
- Plastic waste reduction policies

## Technical Details

- **Frontend**: Pure HTML, CSS, JavaScript
- **Backend**: Hono.js API with blockchain integration
- **Blockchain**: Ethereum mainnet smart contract
- **ENS**: oceanpolicy.eth registry

## Keyboard Shortcuts

- `Ctrl + Enter`: Start simulation from textarea
- Click transaction hashes to copy to clipboard
